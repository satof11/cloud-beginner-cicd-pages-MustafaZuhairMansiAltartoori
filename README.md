# Cloud Beginner CI/CD Pages

This project demonstrates a simple **CI/CD pipeline** using **GitHub Actions** to automatically deploy a static website to **GitHub Pages**.

## Live Website

GitHub Pages Link:
https://satof11.github.io/cloud-beginner-cicd-pages-MustafaZuhairMansiAltartoori/

## Project Files

The repository contains the following files:

* `index.html` – The main webpage that displays my name, course, and information about CI/CD.
* `style.css` – Styling for the website layout and design.
* `script.js` – JavaScript code that adds interactive features to the page.
* `README.md` – Documentation explaining the project.
* `.github/workflows/deploy.yml` – The GitHub Actions workflow that automatically deploys the website.

## How the CI/CD Pipeline Works

1. When code is pushed to the **main branch**, GitHub Actions automatically triggers the workflow.
2. The workflow checks that required files exist (`index.html`, `style.css`, and `script.js`).
3. The website files are uploaded as a deployment artifact.
4. GitHub Actions deploys the website to **GitHub Pages**.
5. The live site updates automatically with the latest changes.

## How to Run the Website Locally

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
